This is my project for my Graphic Processing laboratory, this application renders a complex dungeon environment using the OpenGL library, and allows the user to freely explore the area or to start a dedicated and automated tour of the dungeon showcasing points of interest.
The application is developed in Visual Studio 2022, using the C++ language as well as GLSL for the fragment and vertex shaders. The main porpose of the application is to showcase different lighting and shadowing techniques as well as the texture mapping, rendering and computation of the collision mesh of the dungeon.
All of the assests used inside this project, beside the basic skeleton mesh, were design by me in Blender, including UV mapping and the posing of different assets. Inside the project documentation pdf attached to this repository, is a much more detailed analisys of the application.
